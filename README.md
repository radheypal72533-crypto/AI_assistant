# 🤖 AI Personal Assistant (Flask + OpenAI)

A simple AI-powered web application built using **Flask** and **OpenAI API** that can:

* 💬 Answer user questions
* 📧 Summarize emails
* ⚡ Provide quick and intelligent responses

---

## 🚀 Features

* 🔍 Ask anything (AI chatbot)
* ✉️ Email summarization
* ⚡ Fast API responses
* 🎨 Clean and simple UI
* 🔐 Secure API key using `.env`

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenAI API
* **Environment:** python-dotenv

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/radheypal72533-crypto/AI_assistant.git
cd AI_assistant
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Add your API key

Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key_here
```

### 4️⃣ Run the app

```bash
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000
```
